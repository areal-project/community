# Repositories

This document indexes the repositories that make up the **AReaL** ecosystem.

Anything not listed here is not an official AReaL repository. If you maintain a
project that integrates with AReaL and would like it referenced from the
[Community Projects](#community-projects) section, please open a pull request.

## Core repositories

These repositories form the core of AReaL and are maintained by the AReaL team.

<!-- markdownlint-disable -->

| Name                                                    | Description                                                                            |
| :------------------------------------------------------ | :------------------------------------------------------------------------------------- |
| [AReaL](https://github.com/areal-project/AReaL)         | The main AReaL repository — training framework, examples, and reference algorithms.    |
| [community](https://github.com/areal-project/community) | This repository. Governance, code of conduct, contributing guide, and ecosystem index. |

<!-- markdownlint-restore -->

## Community projects

Projects that build on AReaL or integrate with it. These are maintained by their
respective authors and are not part of the AReaL core release.

## Models & datasets

Released model weights and datasets are published on Hugging Face under the
[inclusionAI](https://huggingface.co/inclusionAI) organization.
