# Code of Conduct

We are the
[PyTorch Foundation Ecosystem project](https://landscape.pytorch.org/?item=training--distributed--areal),
and we are committed to providing a welcoming and safe environment for all participants.
We expect all participants to adhere to the
[PyTorch Foundation Code of Conduct](https://github.com/pytorch-fdn/Foundation/blob/main/PyTorch%20Foundation%20Code%20of%20Conduct.md)
