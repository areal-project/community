# Security Team

The AReaL Security Team is responsible for coordinating the response to security
vulnerabilities and maintaining the security posture of the project.

## Composition

The Security Team consists of all current active Maintainers as listed in
[Maintainers.md](Maintainers.md). All Maintainers are members of the Security Team by
virtue of their role.

## Responsibilities

- Review and respond to security vulnerability reports.
- Coordinate with reporters and the community on security issues.
- Develop and review security patches.
- Manage the security disclosure process.
- Maintain the project's security policy and procedures.

## Communication

Security reports and Security Team discussions are kept private to protect users until a
fix is available. See [SECURITY.md](../SECURITY.md) for the disclosure channels.

## Security Policy

For the complete security policy and reporting procedures, please refer to
[SECURITY.md](../SECURITY.md).
